# react-boilerplate
> Gulp and Browserify Workflow

## Instructions
1. Make sure you have these installed
  - [node.js](http://nodejs.org/)
  - [gulp](http://gulpjs.com/)

2. Run ```npm install``` to install the dependencies

3. Run ```gulp``` command to start the automation
